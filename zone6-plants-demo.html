Zone 6 Plants and Zone Filtering Feature

This document demonstrates the features I added to the Permaculture Tool.

1. Zone Filtering Feature

I added a dropdown menu to filter plants by USDA hardiness zones, with Zone 6 selected by default:

The JavaScript code filters plants based on their growing zones:

function isPlantInZone(plant, zone) {
    // Handle plants with a zone range (e.g., "3-9")
    if (plant.growingZones.includes('-')) {
        const [minZone, maxZone] = plant.growingZones.split('-').map(Number);
        return Number(zone) >= minZone && Number(zone) <= maxZone;
    }
    // Handle plants with individual zones (e.g., "4, 5, 6")
    else if (plant.growingZones.includes(',')) {
        const zones = plant.growingZones.split(',').map(z => Number(z.trim()));
        return zones.includes(Number(zone));
    }
    // Handle plants with a single zone
    else {
        return Number(plant.growingZones) === Number(zone);
    }
}

Type: Perennial Flower

Growing Zones: 3-8

Drought-tolerant native perennial that attracts pollinators and has medicinal properties. Perfect for zone 6 gardens.

Companions: Rudbeckia, Liatris, Ornamental Grasses

Type: Fruit Shrub

Sun: Full sun to partial shade

Acid-loving shrub that produces nutritious berries. Thrives in zone 6 with beautiful fall foliage.

Companions: Strawberry, Thyme, Pine, Azalea

Avoids: Alkaline-loving plants

Native fruit tree with tropical-tasting fruits. Grows naturally in zone 6 forests as an understory tree.

3. How the Application Works

When the application loads:

  1. Zone 6 is pre-selected in the dropdown
  2. Only plants that can grow in zone 6 are displayed
  3. Users can search for specific plants within zone 6
  4. Users can change the zone to see plants for other hardiness zones
  5. Selecting "All Zones" shows all plants regardless of zone

The search functionality works with the zone filter, so users can find specific plants that grow in their selected zone.
